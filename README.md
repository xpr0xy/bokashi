# BOKASHI

A free, accountless browser instrument for exploring traditional Japanese colours, building gradients, extracting palettes, and exporting production assets.

## Capabilities

- 250-colour source-attributed catalogue
- multi-stop linear, radial, and conic gradient maker
- 120 deterministic recipes in an independently scrolling, filterable ledger
- searchable, sortable colour register with copy and local favourites
- monochrome, atmospheric air, six-pattern dither, and deterministic phase-scrubbed cube instruments
- tabbed local source lab for image extraction, feeling mapping, and audio analysis
- interactive HSL/RGB projected 3D colour plane
- catalogue-derived harmony generator, repeatable band field, and WCAG contrast proof
- viewport-bound desktop, tablet, and mobile shell with independently scrolling modes, controls, and browse registers
- PNG, JPG, TIFF, SVG, CSS, and JSON export at configurable dimensions
- local favourites, persistence, and reproducible share URLs

## Development

```bash
npm install
npm run check
npm run dev
```

## Colour data

The traditional colour catalogue is adapted from [xiaohk/nippon-colors](https://github.com/xiaohk/nippon-colors), used under the MIT License. Names and values are presented as a design reference, not as historical scholarship.
