# State inventory

- app: booted, restored, invalid-shared-state
- mode: make, traditional, mono, air, dither, cube, image, plane, catalogue
- recipe: selected, edited, favourited
- stops: 2–5, active stop, position validation
- image: idle, decoding, sampled, invalid type, unreadable
- audio: idle, decoding, analysed, unsupported
- export: closed, ready, rendering, downloaded, failed
- clipboard: idle, copied, denied fallback
- colour plane: idle, dragging, focused point
- persistence: available, unavailable/private mode
- reduced motion: normal, reduced

No state may blank the current artwork. Errors appear beside the initiating control and retain the previous valid output.
